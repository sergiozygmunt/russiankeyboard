Instructions

To type directly with the computer keyboard:

  • Type e= to add a diacritic signe: ë
  • Type zh, ch, sh for ž, č, š
  • Type w for šč
  • Type x for h, for example: sx for: сх
  • Type q after the vowel to add an acute accent (for learning Russian)
